## Design pattern Bridge

Neste repositório tem um caso de uso de uma implementação de código utlizando o padrão Bridge. Este pattern é estrutural e permite dividir uma classe grande ou um conjunto de classes intimamente relacionadas em duas hierarquias separadas — abstração e implementação — que podem ser desenvolvidas independentemente uma da outra. Este exemplo mínimo de código para representar o padrão, consiste em um protótipo de aplicação de Política de cache de um sistema e local de armazenamento associado

Stacks do projeto:

- Spring boot 4.0.4
- Maven
- Java 21
